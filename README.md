Анализ температурных данных и мониторинг текущей погоды

Приложение разработано в рамках домашнего задания по анализу временных рядов и работе с внешними API.  
Проект представляет собой интерактивное Streamlit-приложение для анализа исторических температурных данных и мониторинга текущей температуры через OpenWeatherMap API.

Анализ исторических данных

В рамках анализа реализовано:

- Скользящее среднее и стандартное отклонение (окно 30 дней)
- Поиск аномалий по правилу `rolling_mean ± 2σ`
- Расчёт сезонных статистик (mean, std)
- STL-декомпозиция временного ряда (trend / seasonal / residual)
- Параллельная обработка данных по городам (joblib)

Проведено сравнение последовательного и параллельного анализа.

Мониторинг текущей температуры

- Получение текущей температуры через OpenWeatherMap API
- Реализованы:
  - синхронный запрос (requests)
  - асинхронный запрос (aiohttp)
- Определение, является ли текущая температура нормальной для сезона
- Эксперимент по сравнению времени выполнения sync и async запросов

Интерфейс Streamlit

Приложение позволяет:

- Загружать CSV-файл с историческими данными
- Выбирать город из списка
- Вводить API-ключ OpenWeatherMap
- Выбирать режим запроса (sync / async)
- Просматривать:
  - описательную статистику
  - временные ряды температур с аномалиями
  - сезонные профили
  - STL-декомпозицию
  - текущую температуру и её статус (норма / аномалия)

Все графики реализованы с помощью Plotly и являются интерактивными.
